# Quick Compressor

動画ファイルのファイルサイズを劇的に軽量化するための、Windows右クリックメニュー統合型の超高速・高画質動画圧縮ツールです。

NVIDIA製GPUのハードウェアエンコード（NVENC）およびAMD製GPUのハードウェアエンコード（AMF）を最大限に活用し、画質の劣化を最小限に抑えながら超高速で動画を圧縮します。Discordや各種SNSへのアップロード時、動画の保管容量を節約したい時など、素早く簡単に動画を軽量化したい場面に最適なソフトウェアです。

---

## 🚀 主な機能

- **超高速 GPU ハードウェアエンコード**
  NVIDIA NVENC および AMD AMF (H.264, HEVC/H.265, AV1対応) を利用し、CPUに負荷をかけずに短時間で動画を軽量化します。デコード（読み込み）にもGPUを使用するため、非常に高速です。
- **目標ファイルサイズ指定 (容量優先モード)**
  「50MB以内に収める」「10MBにする」など、目標とするファイルサイズを指定するだけで、ツールが自動的に最適なビットレートを計算し、指定容量に収まるように圧縮します。
- **右クリックメニューからの即時実行 (プリセット連携)**
  動画ファイルを右クリックして表示されるプリセット（「Discord用」「X(旧Twitter)用」など）を選ぶだけで、即座に最適な設定で軽量化処理が開始されます。
- **直感的な GUI とプリセット管理**
  解像度、フレームレート、画質（CQP）などの細かい設定を専用のGUIから直感的に変更できます。また、よく使う設定を保存したり、アプリ内からプリセットの名前を変更・削除できる管理機能（GUI）も備えています。
- **自動アップデート確認機能**
  アプリ起動時に GitHub Releases の最新バージョンを自動的に確認します。最新版があればワンクリックでブラウザからダウンロードページへアクセス可能です（API制限対策として、確認は24時間に1回のみ通信を行います）。

## 💻 動作要件

### 一般ユーザー（配布用インストーラーを使用する場合）
- **OS**: Windows 10 / 11 (64bit)
- **GPU**: NVIDIA製グラフィックボード（NVENC対応）または AMD製グラフィックボード（AMF対応）
- **依存ソフト**: 不要（PythonやFFmpegのインストール・設定は一切不要です）

### 開発者・ソースから実行する場合
- **環境**: Python 3.12 以降
- **依存ソフト**: FFmpeg および FFprobe（システム環境変数 `PATH` に通っているか、アプリフォルダ内に配置されていること）

## 📦 インストール方法（一般ユーザー向け）

1. [GitHub Releases](https://github.com/LunaFleuret/File-Converter-MOD/releases) から最新の `QuickCompressor_Setup.exe` をダウンロードします。
2. ダウンロードしたインストーラーを実行します。画面の指示に従うだけで自動的にインストールが完了します。
3. インストールが完了すると、自動的にエクスプローラーの右クリックメニューへ登録されます。

### アンインストール
Windowsの「設定」＞「アプリ」＞「インストールされているアプリ」から「Quick Compressor」を選択してアンインストールしてください。右クリックメニューへの登録も自動的に削除されます。

## 🛠️ 開発者向けセットアップとビルド手順

ソースコードから直接実行または自分でビルド・配布を行う場合の手順です。

### 1. 手動での右クリックメニュー登録
ソースコードから直接起動する構成で右クリックメニューに登録します：
```bash
python register_menu.py --register
```
※ 単純に `python register_menu.py` を実行すると、登録・解除を行うための専用画面が起動します。解除する場合は `python register_menu.py --unregister` を実行します。

### 2. インストーラー（setup.exe）のビルド方法
自分でビルドして他の人に配布するインストーラーを作成する手順です：

1. **必要なツールの準備**:
   - `pip install pyinstaller` で PyInstaller をインストールします。
   - [Inno Setup](https://jrsoftware.org/isdl.php) をインストールします。
2. **実行ファイルの作成**:
   - `build.bat` を実行します。自動的に `pyinstaller` が走り、`dist` フォルダに `QuickCompressor.exe` が作成されます。
3. **FFmpegの同梱**:
   - `dist\bin\` フォルダを作成し、Windows版の `ffmpeg.exe` と `ffprobe.exe` をコピーして配置します。
4. **インストーラーのコンパイル**:
   - Inno Setup で `build_installer.iss` を開き、コンパイルを実行します。`Output\` フォルダに配布用の `QuickCompressor_Setup.exe` が生成されます。

## 💡 使い方

### 右クリックから素早く圧縮（プリセット使用）
1. 圧縮したい動画ファイル（`.mp4`, `.mkv`, `.mov` など）を右クリックします。
2. Windows 11 の場合は「その他のオプションを表示」をクリックし、**「Quick Compressor」** の中から「Discord用」などの目的に合ったプリセットを選択します。
3. 自動的に設定が読み込まれ、超高速で圧縮が開始されます。

### 詳細設定からの圧縮・プリセット管理
1. 右クリックメニューの「Quick Compressor」から **「設定...」** を選択するか、プログラムを直接起動すると、GUI画面が表示されます。
2. 画面内で、出力コーデック、フレームレート、解像度、画質（品質優先 / 容量優先）などを細かく調整できます。
3. **「⚡ 圧縮開始」** ボタンをクリックすると変換が開始されます。
4. **「🎁 プリセット作成」** ボタンを押すと、現在の設定を新しいプリセットとして保存し、次回から右クリックで使えるようになります。
5. **「📋 プリセット管理」** ボタンからは、登録されているすべてのプリセットの名前変更や削除が行えます。
