# Quick Compressor

Discord（10MB）や、Steam Chat（30MB）など、極めて小さいファイル制限が求められるコミュニティ向けの圧縮ソフト

通話中でも流れを止めることなくクリップの共有が出来ます。

動画ファイルのファイルサイズを軽量化するための、Windows右クリックメニュー統合型の動画圧縮ツール。

Discordや各種SNSへのアップロード時、動画の保管容量を節約したい時など、素早く簡単に動画を軽量化したい場面に最適。

---

## 🚀 主な機能

- **目標ファイルサイズ指定 (容量優先モード)**
  「50MB以内に収める」「10MBにする」など、目標とするファイルサイズを指定するだけで、ツールが自動的に最適なビットレートを計算し、指定容量に収まるように圧縮します。
- **右クリックメニューからの即時実行 (プリセット連携)**
  動画ファイルを右クリックして表示されるプリセット（「Discord用」「X(旧Twitter)用」など）を選ぶだけで、即座に最適な設定で軽量化処理が開始されます。

---

## 💻 動作要件

### 一般（配布用インストーラーを使用する場合）

- **OS**: Windows 10 / 11 (64bit)
- **GPU**: NVIDIA製グラフィックボード（NVENC対応）または AMD製グラフィックボード（AMF対応）
- **依存ソフト**: 不要

### 開発者・ソースから実行する場合

- **環境**: Python 3.12 以降
- **依存ソフト**: FFmpeg および FFprobe（システム環境変数 `PATH` に通っているか、アプリフォルダ内に配置されていること）

## 📦 インストール方法（一般ユーザー向け）

1. [GitHub Releases](https://github.com/LunaFleuret/File-Converter-MOD/releases) から最新の `QuickCompressor_Setup.exe` をダウンロードします。
2. ダウンロードしたインストーラーを実行します。画面の指示に従うだけで自動的にインストールが完了します。
3. インストールが完了すると、自動的にエクスプローラーの右クリックメニューへ登録されます。

### アンインストール

Windowsの「設定」＞「アプリ」＞「インストールされているアプリ」から「Quick Compressor」を選択してアンインストールしてください。右クリックメニューへの登録も自動的に削除されます。

---

## 💡 使い方

### 右クリックから素早く圧縮（プリセット使用）

1. 圧縮したい動画ファイル（`.mp4`, `.mkv`, `.mov` など）を右クリックします。
2. Windows 11 の場合は「その他のオプションを表示」をクリックし、**「Quick Compressor (プリセット)」** の中から「Discord用」などの目的に合ったプリセットを選択します。
3. 自動的に設定が読み込まれ、圧縮が開始されます。

### 詳細設定からの圧縮・プリセット管理

1. 右クリックメニューの「Quick Compressor」から **「設定...」** を選択するか、プログラムを直接起動すると、GUI画面が表示されます。
2. 画面内で、出力コーデック、フレームレート、解像度、画質（品質優先 / 容量優先）などを細かく調整できます。
3. **「⚡ 圧縮開始」** ボタンをクリックすると変換が開始されます。
4. **「🎁 プリセット作成」** ボタンを押すと、現在の設定を新しいプリセットとして保存し、次回から右クリックで使えるようになります。
5. **「📋 プリセット管理」** ボタンからは、登録されているすべてのプリセットの名前変更や削除が行えます。
