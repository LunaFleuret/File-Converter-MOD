# GPU Video Converter (File Converter-Mod)

NVIDIA GPU (NVENC) を使用して、動画を高速に圧縮・変換する Windows 右クリックメニュー統合型のツールです。
「元の File Converter では 60fps から 30fps への変更や、GPU を使った細かい圧縮率（ビットレート・クオリティ）の調整ができない」という課題を解決するために作成されました。

## 主な機能
- **超高速 GPU エンコード**: NVIDIA NVENC (H.264, HEVC, AV1) を使用し、グラフィックボード (GTX 1660 Super 等) の性能をフルに活かしてエンコードします。
- **解像度調整**: スライダーで解像度を 25% 〜 100% に縮小可能。
- **画質（クオリティ）調整**: CQP (Constant Quantization Parameter) スライダーで画質とファイルサイズのバランスを直感的に調整。
- **フレームレート変更**: 元のまま、または 24fps / 30fps / 60fps への変換に対応。
- **音声ミュート機能**: 音声トラックを除去して動画のみにするオプション。
- **Windows 右クリックメニュー連携**: 動画ファイルを右クリックして「GPU動画変換で圧縮」から一発で起動できます。

## 動作要件
- Windows OS
- NVIDIA グラフィックボード (GTX 1660 Super など、NVENC 対応のもの)
- Python 3.x
- FFmpeg (環境変数 `PATH` に登録されている、またはシステムで利用可能であること)

## セットアップ手順

### 1. 右クリックメニューへの登録
本ツールを右クリックメニューに登録します。管理者権限は不要です（ユーザー環境のレジストリに登録されます）。

1. このフォルダでコマンドプロンプトや PowerShell を開きます。
2. 以下のコマンドを実行します：
   ```bash
   python register_menu.py --register
   ```
   または、単純に `python register_menu.py` を実行すると、登録・解除を行うためのシンプルな画面が起動します。

### 2. アンインストール（メニューの解除）
右クリックメニューから本ツールを削除したい場合は、以下のコマンドを実行します：
```bash
python register_menu.py --unregister
```

## 使い方
1. 変換したい動画ファイル（`.mp4`, `.mkv`, `.mov` など）を右クリックします。
2. Windows 11 の場合は「その他のオプションを表示」をクリックし、**「GPU動画変換で圧縮」**を選択します。
3. 変換設定画面（GUI）が起動するので、解像度、画質、フレームレート、コーデックなどを調整します。
4. **「変換開始」**ボタンをクリックすると、GPU による高速エンコードが開始されます。
5. 変換後のファイルは、元のファイルと同じフォルダに `_converted` というサフィックスが付いて保存されます。
